### 👨‍💻 Engineering & Data Focus

I am a software developer with a strong focus on Data Science, Artificial Intelligence, and building efficient tooling. Whether it's publishing Python packages to PyPI, building modern web interfaces, or engineering custom hardware solutions, I thrive on solving complex technical problems from end to end.

- 📦 **Shipping Code:** I am the creator of [**cinexg-eda**](https://pypi.org/project/cinexg-eda/), a Python package for automated Exploratory Data Analysis.
- 🎓 **Community & Education:** I teach Python programming and technical problem-solving on [**my YouTube channel**](https://youtube.com/@cinexg).
- ⚙️ **Hardware Tinkering:** I actively build custom DIY electronics, including a fully customized, water-cooled Arduino station.
- 🌐 **Web Architecture:** I build highly performant, custom-built web applications using **Next.js** and vanilla web technologies.
